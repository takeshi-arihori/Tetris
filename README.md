# Tetris

##### HTML・CSS・JavaScriptを使ってテトリスを作成しました。


#### 基本ルール
・ 10 × 20 のフィールド
・ フィールド上部からランダムにテトリミノが落下
・ テトリミノが埋まったラインは消え、消したライン数でスコアを加算

|    ライン数     |     スコア      |
| ------------- | ------------- |
|       1       |       40       |
|       2       |       100       |
|       3       |       400       |
|       4       |       1200       |


・ スコアに応じてテトロミノの落下速度が早くなる
・ レベル8, 9に到達するとBGMが変化する

|     レベル     |       スコア      |   落下速度(1ライン)  |
| ------------- | --------------- |  ---------------  |
|       1       |       3000      |        0.9s       |
|       2       |       6000      |        0.8s       |
|       3       |       9000      |        0.7s       |
|       4       |       12000     |        0.6s       |
|       5       |       15000     |        0.5s       |
|       6       |       18000     |        0.4s       |
|       7       |       21000     |        0.3s       |
|       8       |       24000     |        0.2s       |
|       9       |       27000     |        0.1s       |
